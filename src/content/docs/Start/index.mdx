本人曾使用多个博客框架（虽然你现在看到的[Starlight](https://starlight.astro.build/zh-cn)是一个文档框架），比如[xLog](https://xlog.app)、[WordPress](https://cn.wordpress.org/)、[Hexo](https://hexo.io/)、[Halo](https://halo.run/)、[Fuwari](https://github.com/saicaca/fuwari)、[Typecho](https://typecho.org/)、[VitePress](https://vitepress.vuejs.org/)、自己用Qwik手搓

但我在某一天翻阅[Astro文档](https://docs.astro.build/zh-cn)的时候发现它的文档做的很好（甚至有ToDo），欸！就这样发现了名为[Starlight](https://starlight.astro.build/zh-cn)的文档框架，在我看来，它比[VitePress](https://vitepress.vuejs.org/)更好看，总之现在就用啦！
